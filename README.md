# ML-DS-para-Peliculas
¿Nunca has visto una película que tenia buena pinta y no te gusto o viceversa? Este proyecto viene dado por una falta real para saber si una película me gustará mas o menos. Para ello dado que en la pagina web de palomitacas tenia todas las películas (una 330) que he visto en los últimos años junto con mi valoración me plantee este proyecto.
